# ?? DnD Telegram RPG

AI-powered RPG game in Telegram using Gemini.

## ?? Features
- Classes (Warrior, Mage, Archer)
- Combat system
- Inventory
- Multiplayer (party)
- AI quests

## ?? Setup

1. Clone repo
2. Create .env from .env.example
3. Install dependencies:
   pip install -r requirements.txt

## ?? Run locally

uvicorn app.main:app --reload

## ?? Deploy
Deploy to Railway and add ENV variables.
